About — Taku Ora Haerenga

This project is a scroll-driven storytelling experience created by Nigel Murch. It documents a personal journey through hardship, grief, and recovery. The site uses simple HTML, CSS and vanilla JavaScript along with GSAP for animation.

About the author

Nigel created this narrative to be candid and to offer connection for readers who may be navigating similar struggles. If you'd like to contact him, use the contact page linked in the site navigation.
